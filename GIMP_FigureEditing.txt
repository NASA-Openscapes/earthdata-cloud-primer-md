2GIMP Editing
- Copy/Paste image into GIMP
- Crop to image boundaries
- Add border: Filters > Decor > Add Border... > {Border X size: 2, Border Y size: 2, Border color: {HTML notation: 7f7f7f}, Delta value on color: 1}
- Add box: Select Rectangle or Ellipse Select Tool > Drag 'box' over area to highlight > Ensure Foreground color is set to Red > Select Edit on the toolbar > Select Stroke Selection... > Line width = 3.0 px > Click Stroke
